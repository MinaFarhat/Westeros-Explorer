# Westeros Explorer

**Westeros Explorer** is a Flutter application inspired by *Game of Thrones* for exploring images, episodes, and character details. The project demonstrates practical usage of **Flutter UI**, **image processing**, and **search functionality**, providing an interactive and efficient way to browse and manage GoT content.

## Features

- User-friendly interface to browse and explore Game of Thrones content  
- Search images by similarity, color, size, or date  
- Crop and save images  
- Responsive Flutter UI  

## Getting Started

This project is built with Flutter and demonstrates interactive UI design and image search functionality.

### Prerequisites

- Flutter SDK installed  
- Dart  
- An IDE like **VS Code** or **Android Studio**
